About me

Who?

My name is Aleksander Madland Stapnes and I'm a freelance web developer.

I write about programming and related subjects.

I've programmed in Python, Ruby and Javascript and dabbled in others before finding Clojure and Clojurescript, which are my current preferred languages. I consider myself a polyglot programmer, and will program in pretty much anything if it makes sense to do so. Bonus points if it's functional though.
